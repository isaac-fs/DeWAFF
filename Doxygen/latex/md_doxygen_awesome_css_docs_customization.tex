\doxysubsection{CSS-\/\+Variables}\label{md_doxygen_awesome_css_docs_customization_autotoc_md1}
This theme is highly customizable because a lot of things are parameterized with CSS variables.

Just to give you an idea on how flexible the styling is, click this button\+:

Alter theme\doxysubsubsection{Setup}\label{md_doxygen_awesome_css_docs_customization_autotoc_md2}
It is recommended to add your own {\ttfamily custom.\+css} and overwrite the variables there\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{HTML\_EXTRA\_STYLESHEET  = doxygen-\/awesome.css custom.css}

\end{DoxyCode}


Make sure to override the variables in the correct spot. All variables should be customized where they have been defined, in the {\ttfamily html} tag selector\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{html \{}
\DoxyCodeLine{    /* override light-\/mode variables here */}
\DoxyCodeLine{\}}

\end{DoxyCode}


For dark-\/mode overrides you have to choose where to put them, depending on wether the dark-\/mode toggle extension is installed or not\+:


\begin{DoxyItemize}
\item dark-\/mode toggle is installed\+: \`{}\`{}\`{}css html.\+dark-\/mode \{ /$\ast$ define dark-\/mode variable overrides here if you DO use doxygen-\/awesome-\/darkmode-\/toggle.\+js $\ast$/ \} \`{}\`{}\`{}
\item dark-\/mode toggle is {\bfseries{NOT}} installed. The dark-\/mode is enabled automatically depending on the system preference\+: \`{}\`{}\`{}css @media (prefers-\/color-\/scheme\+: dark) \{ html\+:not(.light-\/mode) \{ /$\ast$ define dark-\/mode variable overrides here if you DON\textquotesingle{}T use doxygen-\/awesome-\/darkmode-\/toggle.\+js $\ast$/ \} \} \`{}\`{}\`{}
\end{DoxyItemize}\doxysubsubsection{Available variables}\label{md_doxygen_awesome_css_docs_customization_autotoc_md3}

If you miss a configuration option or find a bug, please consider {\texttt{ opening an issue}}!\doxysubsection{Doxygen generator}\label{md_doxygen_awesome_css_docs_customization_autotoc_md4}
The theme overrides most colors with the {\ttfamily -\/-\/primary-\/color-\/$\ast$} variables.

But there is a few small images and graphics that the theme cannot adjust or replace. To make these blend in better with the rest, it is recommended to adjust the {\texttt{ doxygen color settings}} to something that matches the chosen color-\/scheme.

For the default color-\/scheme, these values work out quite well\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\# Doxyfile}
\DoxyCodeLine{HTML\_COLORSTYLE\_HUE    = 209}
\DoxyCodeLine{HTML\_COLORSTYLE\_SAT    = 255}
\DoxyCodeLine{HTML\_COLORSTYLE\_GAMMA  = 113}

\end{DoxyCode}

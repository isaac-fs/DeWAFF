Generate a meshgrid from X and Y unidimensional coordinates Example: xRange = [0,3[ and yRange = [0,3[ will return the following X and Y coordinates.

X =
[0, 0, 0;
1, 1, 1;
2, 2, 2]
Y =
[0, 1, 2;
0, 1, 2;
0, 1, 2]

Wich would form the following mesh grid

(X,Y) =
[(0,0) (0,1), (0,2);
(0,1) (1,1), (2,1);
(2,0) (2,1), (2,2)]
